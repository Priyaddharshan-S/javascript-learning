# DOM
The Document Object Model (DOM) is the browser’s in-memory tree representation of a webpage. Each element, attribute, and piece of text in the document becomes a node in this tree, allowing programming languages like JavaScript to access, traverse, and manipulate the content, structure, and style of a webpage dynamically.

### Interview style definition:
* The Document Object Model (DOM) is the browser’s in-memory tree representation of a webpage used for accessing, traversing and manipulate the content, structure and style of a webpage dynamically.
